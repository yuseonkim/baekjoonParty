9장 다형성

1. 다형성이란
-상향 형변환 : 자식클래스를 가리키는 포인터가 부모 클래스를 가리키도록 형변환
	Dog dog;
	Animal *pa;
	pa = &dog;
-하향 형변환 : 부모클래스를 가리키는 포인터가 자식 클래스를 기리키도록 형변환
	BaseClass *pb = new DerivedClass();
	DerivedClass *pb = (DerivedLass *)pb;
-함수의 매개변수 : 부모클래스로 하는 게 좋음

2. 가상함수
 1) 가상함수
- 부모 클래스의 포인터로 호출되더라고 자식 클래스의 재정의된 함수가 호출되게 하는 함수
2) 동적 바인딩 
- 바인딩(함수호출-함수의몸체 연결)이 실행시간에 호출되는 함수를 결정하는 것
- 다형성 구현 가능
- 가상함수는 동적 바인딩으로 호출
3)참조자와 가상함수
- 포인터 대신 참조자 써도 결과 동일

4. 가상 소멸자
: 가상함수 소멸자 쓸  때, 부모클래스의 소멸자를 가상함수로 선언!!
!!자식 클래스의 소멸자가 호출된 후에는 항상 부모 클래스의 소멸자가 
호출된다는 것을 잊지 말자!!

5.순수가상함수
: 헤더만 존재하고 몸체는 없는 함수
	virtual 반환형 함수이름(매개변수 리스트) = 0;
1)추상클래스 
- 순수가상함수를 하나라도 가지고 있는 클래스, 객체 생성 불가
- 추상적인 개념, 클래스들 사이의 인터페이스 나타낼 때 사용
- 멤버함수의 원형만을 정의, 구현은 자식 클래스에서
- 추상클래스를 상속받는 자식클래스는 반드시 순수가상함수를 구현!!
- 추상클래스에 포함된 모든 가상함수를 재정의 해야만 함
