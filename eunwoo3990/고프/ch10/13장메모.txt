13장 입출력과 파일처리

1. 파일입출력
1) 스트림이란 :  입력과 출력을 바이트들의 흐름으로 생각하는 것
 - 장치 독립성
2) <<과 >>을 이용한 파일 입출력
 - ifstream : 파일을 통하여 입력할 때 사용되는 스트림
 - 파일에 연결하려면 open() 멤버 함수 사용
 - ofstream : 파일에 자료를 저장하는 출력 스트림
3) 멤버 함수를 이용한 파일 입출력
 - 하나의 문자를 읽는 데 get(), 하나의 문자를 출력하는데 put()
 - 파일의 끝을 알아내려면 eof() 함수
4) 출력 형식 지정
 - setf() 함수 : 출력 플래그 설정
 - unsetf : 플래그 해제

2. 텍스트 파일과 이진 파일
1) 이진 파일 입출력 : fstream의 read()와 write()
write(char *buffer, int count);

3. 순차 파일과 랜덤 파일
 - 순차 접근 : 현재 파일 닫고 다음 파일 열고
 - 임의 접근 : 파일의 어느 위치에서든 읽기와 쓰기 가능
